# 🏥 Healthcare Management System

A responsive Healthcare Management System frontend developed using **HTML, CSS, and JavaScript**. The project provides an intuitive interface for patients to explore medical services, learn about doctors, book appointments, and contact the healthcare facility.

## ✨ Features

### 🏠 Home Page

* Professional healthcare landing page
* Responsive hero section
* Easy navigation across all sections

### 👨‍⚕️ Our Doctors

* Doctor profile cards
* Specialization details
* "Know More" functionality for additional information
* Clean and modern card-based design

### 🏥 Our Specialities

* Display of available medical specialties
* Expandable information sections
* User-friendly layout for quick access to details

### 📅 Appointment Booking

* Patient appointment booking form
* Input validation using JavaScript
* Doctor selection and appointment scheduling
* Email confirmation integration using EmailJS

### 📞 Contact Us

* Contact form with validation
* EmailJS integration for message delivery
* Easy communication between patients and healthcare providers

### 📱 Responsive Design

* Mobile-friendly interface
* Tablet and desktop compatibility
* Professional healthcare-themed UI
